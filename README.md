# Gracie Bites – Premium Restaurant Landing Page
A modern, high-conversion landing page built for a premium African culinary brand. This project focuses on delivering a "digital-first" brand experience with smooth interactive elements and a mobile-responsive grid.

✨ Key Features
Polished UI/UX: A clean, high-contrast design optimized for food photography and brand storytelling.

Fully Responsive: Built with a Mobile-First mindset using Bootstrap 5 to ensure a seamless experience on smartphones and tablets.

Interactive Animations: Integrated AOS (Animate On Scroll) for elegant, fluid transitions as the user explores the menu.

Optimized Performance: Fast-loading assets and clean HTML/CSS structure for better SEO and user retention.

___________________________________________________________________________________________
🛠️ Tech Stack
HTML5 – Semantic structure.

CSS3 – Custom styling and layout refinements.

Bootstrap 5 – Responsive grid system and UI components.

AOS Library – Scroll-triggered animations.

GitHub Pages – Deployment and hosting.
